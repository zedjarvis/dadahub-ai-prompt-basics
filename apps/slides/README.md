# Workshop Slides

Slidev deck for the Dada Hub Residency Week 3 workshop.

## Run

From the repo root:

```bash
pnpm dev:slides
```

Then open:

- `http://localhost:3030`

## Source

- main deck: [slides.md](./slides.md)

## Deck Coverage

The slides are structured to support:

1. API access
2. Prompt engineering basics
3. Simple chatbot architecture
4. Embeddings and semantic search
5. Cost optimization
6. Self-directed block + homework

## Notes

- Theme: `@slidev/theme-bricks`
- The deck is paired with the live chatbot demo in `apps/web`
- Built with [Slidev](https://sli.dev/)
